# Opticalia Veravisión - Migración a Material Design 3

Este proyecto migra la interfaz original de Opticalia Veravisión a un front-end funcional usando **Material Design 3** y la librería oficial **@material/web**.

## Ejecutar localmente

```bash
npm install
npm run dev
```

Luego abre la URL que muestre Vite en la terminal.

## Componentes Material usados

- `md-filled-button`, `md-outlined-button`, `md-text-button`: acciones principales y secundarias.
- `md-icon-button` y `md-icon`: acciones rápidas e iconografía.
- `md-outlined-text-field`: búsqueda global y búsqueda de clientes.
- `md-filter-chip` y `md-assist-chip`: filtros, estados y categorías.
- Tokens `--md-sys-color-*`: colores, superficies, bordes y jerarquía visual.

## Funcionalidades conservadas

- Navegación entre Panel Principal, Clientes, Seguimiento, Insights y Redes Sociales.
- Filtro de clientes por texto y tipo.
- Vista de detalle de cliente.
- Cambio de estado de tareas.
- Toasts/notificaciones visuales para confirmar acciones.
- Diseño responsive para pantallas pequeñas.
